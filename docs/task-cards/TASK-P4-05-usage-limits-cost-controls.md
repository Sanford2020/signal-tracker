# TASK-P4-05: Usage Limits and Cost Controls

## Goal

Track workspace usage and enforce simple monthly limits for AI extraction and source checks.

## Scope

- Add usage event storage.
- Record AI extraction and source check usage.
- Expose workspace usage summary.
- Enforce configurable monthly limits.

## Acceptance

- Usage summaries show monthly totals.
- AI extraction records usage and respects limit.
- Source checks record usage and respect limit.
- Default limits do not disrupt existing MVP flows.
