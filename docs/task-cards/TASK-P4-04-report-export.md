# TASK-P4-04: Report Export

## Goal

Export daily and weekly intelligence reports as Markdown and PDF.

## Scope

- Add report rendering service.
- Add Markdown and PDF endpoints for daily and weekly briefings.
- Add export links to the briefing UI.

## Acceptance

- Daily and weekly Markdown exports return readable sections.
- Daily and weekly PDF exports return valid PDF bytes.
- Frontend links expose exports for selected report mode.
