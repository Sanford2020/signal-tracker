# TASK-P4-02: Team Notes and Ownership

## Goal

Add basic team collaboration around intel files: owner assignment, review metadata, and comments.

## Scope

- Add owner and review fields to intel files.
- Add intel file comments.
- Add APIs to update collaboration metadata and add/list comments.
- Show owner/review/comments on the intel file detail page.

## Acceptance

- A file can have an owner and review note.
- Comments are persisted with author metadata.
- Workspace-scoped detail still protects files from other workspaces.
- Backend and frontend validation passes.
